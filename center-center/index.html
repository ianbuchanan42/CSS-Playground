Center With CSS

Display:flex

.flex-container { display: flex; align-items: center; justify-content: center; } #flex-center { width: 10px; height: 10px; }

Centering with Flexbox involves using a container that is set to display "flex". Then the container must have "align-items" and "justify-content" set to "center".

Transform: translateX(-50%) translateY(-50%);

.relative-container { position: relative; } #transform { width: 10px; height: 10px; position: absolute; top: 50%; left: 50%; transform: translateX(-50%) translateY(-50%); }

Using transform involves having to set the parent or container to "relative" (or any other position value other than static, otherwise the child will relate itself to the closest set ancestor). Then the element intended for centering is set to "absolute" in relation to the parent container. Allowing "top" and "left" to be set to "50%"", putting the corder of the child in the center of the container. Then using "transform" we set "translateX" and "translateY" at "-50%"" centers the child in relation to itself.

Transform: translate(-50%,-50%);

#transform-shorthand { width: 10px; height: 10px; position: absolute; top: 50%; left: 50%; transform: translate(-50%, -50%); }

Using the short hand of "translate"

Display:grid

.grid-container { display: grid; place-items: center; } #grid-center { width: 10px; height: 10px; }

Using grid we can using "place-items" set to "center"

Margin: auto

#margin-center { position: absolute; width: 10px; height: 10px; top: 0; left: 0; right: 0; bottom: 0; margin: auto; }

Using "margin" "auto" works as long as the element is set to "absolute" position and top, left, right and bottom art set to 0
